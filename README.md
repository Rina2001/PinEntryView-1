# PinEntryView
Pin/OTP entry field library for Android

![](demo.gif)


Usage example:

```xml
<com.hm60.pinentryview.PinEntryView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            app:line_color="#ff0000"
            app:number_count="5"
            app:text_color="#1f1f1f"/>
```
